# Examples

## Quick Demo

Try translating the demo page:

```bash
cd examples
../auto-translate.sh demo.html "en-US,ja-JP"
```

Result:
- `demo.html__multilang__en-US/demo.html` - English version
- `demo.html__multilang__ja-JP/demo.html` - Japanese version
